Information to the Graph

The Graph is based on the weather forecast of the "Open-Meteo" API (https://open-meteo.com/).

There is no grant that all the data is correct. This project should only display the weather information.

The table on the left side shows how to understand the mm scale. One millimeter of is corresponding to one liter of Water on one square meter in one hour.
